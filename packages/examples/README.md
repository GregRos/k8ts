# @k8ts/examples

This package contains examples demonstrating the usage of k8ts and related packages.

## Structure

Each subdirectory in `examples/` contains a separate example:

- `dummy/` - A simple "Hello World" example

## Running Examples

Build the package:

```bash
yarn build
```

Run a specific example:

```bash
node dist/examples/dummy/index.js
```

## Development

Install dependencies:

```bash
yarn install
```

Build:

```bash
yarn build
```

Clean build artifacts:

```bash
yarn clean
```
